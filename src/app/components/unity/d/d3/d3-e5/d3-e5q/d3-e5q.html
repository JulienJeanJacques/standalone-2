Qu’appelle-t-on Zodiaque ?
1) Le Zodiaque est une bande circulaire parallèle à l’équateur céleste qui contient les constellations dites zodiacales.
2) Le Zodiaque est un ensemble de constellations dont le nom évoque un être vivant.
3) Le Zodiaque est l’ensemble des constellations qui se répartissent le long de la Voie Lactée.
4) Le Zodiaque est une bande circulaire faisant le tour de la sphère céleste contenant toutes les constellations que le Soleil traverse successivement au cours d’une année.
