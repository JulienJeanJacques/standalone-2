Parmi les caractéristiques communes au mouvement de toutes les planètes, l’une d’elles est fausse ; laquelle ?
1) Les planètes tournent toutes dans le même sens autour du Soleil.
2) Les orbites de toutes les planètes sont approximativement dans un même plan.
3) Une planète tourne autour du Soleil d’autant plus vite qu’elle en est proche.
4) La rotation de toutes les planètes autour de leur axe s’effectue dans le même sens que leur révolution autour du Soleil.
