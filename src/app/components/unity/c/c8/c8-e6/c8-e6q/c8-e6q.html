Parmi les quatre affirmations concernant la visibilité de Vénus, une seule est exacte, laquelle ?
1) Comme pour toutes les planètes du Système solaire, l’opposition est le meilleur moment pour observer Vénus.
2) Si Vénus est visible à l’œil nu un soir, elle le sera encore le lendemain matin, juste avant le lever du jour.
3) Lorsque Vénus se retrouve dans la même direction que la Pleine Lune, elle est nécessairement en phase pleine.
4) L’élongation maximale présente l’un des moments le plus favorable pour observer Vénus.
